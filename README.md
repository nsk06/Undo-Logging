#### This directory contains code for implementation of undo logging and recovery from it.
##### Running Instructions for Logs
- python3 20171203_1.py filename x
- Here filename is the testfile for log generation and x is the number of instructions that are executed at a time.
##### Running Instructions for Recovery
- python3 20171203_2.py filename
-  Here filename is the testfile for recovery from log (following non quiscient checkpointing)
